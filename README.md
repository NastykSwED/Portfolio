# Run Locally

<br/>

Clone the project

```bash
  git clone https://github.com/NastykSwED/Portfolio
```

Go to the project directory

```bash
  cd Portfolio
```

Install dependencies

```bash
 pnpm install
```

Start the server

```bash
  pnpm run dev
```
